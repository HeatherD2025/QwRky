# QwRky

<img src="./QwRky_logo.webp" alt="QwRky logo" width="300"/>

### Created By

- [Heather DeLiso](https://github.com/HeatherD2025)

A science themed site that combines news and images from around the world and beyond.

## Key Features

- **Api Integration**: uses a combination of three apis as of 1/2026, with more to be added
- **Polished Dashboard**: Visually appealing and organized feeds

## Tech

### Frontend

- React.js with Vite
- React Bootstrap for responsive layout
- React Router for navigation
- Redux for state management

### Key Dependencies

- `react-bootstrap` - UI components
- `react-redux` - State management
- `react-router-dom` - Routing
